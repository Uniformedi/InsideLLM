Active Directory Integration

Join this VM to your AD domain so PAM-backed services (SSH, Cockpit, sudo) can authenticate against your DC. Independent of LDAP-bind auth used by the InsideLLM admin pages.
One-time, host-level operation. Joining writes machine credentials, modifies /etc/sssd, restarts the auth daemon, and registers a computer object in AD. The domain admin password you submit is consumed by a host-side script and immediately deleted; never persisted by the Hub.
