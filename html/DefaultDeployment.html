InsideLLM — Default Deployment Reference

Every Terraform input variable and the default used when no override is provided in terraform.tfvars.

Platform: v3.1.0 Generated: 2026-04-16 Variables: 142
These are the defaults baked into terraform/variables.tf. The Setup Wizard surfaces the most important variables and emits a terraform.tfvars file; operators can override any variable below by adding it to that file. Sensitive inputs — API keys, passwords, secrets — have no safe default and are flagged secret — set in tfvars.
