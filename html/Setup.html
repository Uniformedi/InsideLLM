SSO Provider

SSO applies to all services: Open WebUI (chat), LiteLLM (API), Grafana (dashboards), and the Admin Center. A single app registration in your IdP is shared across all services.

Redirect URIs to register in your IdP:
  • https://<your-host>/oauth/oidc/callback (Open WebUI)
  • https://<your-host>/grafana/login/generic_oauth (Grafana)
  • https://<your-host>/litellm/sso/callback (LiteLLM)
  • https://<your-host>/auth/callback (Admin Command Center)
Admin Portal Authentication: When AD domain join is enabled without a cloud SSO provider, the Admin Command Center is protected by Active Directory credentials. Users sign in with their AD username and password. Access is restricted to members of the configured admin groups (default: Domain Admins).

Group-to-Team Mapping

Map your SSO groups to LiteLLM teams. Each group gets its own budget, rate limits, and model access. Members are automatically assigned on login.

The JWT claim name that contains group membership. Usually "groups" for both Azure AD and Okta.
