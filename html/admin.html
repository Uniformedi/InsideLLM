About this view
The governance framework is the structured document that defines your organization's AI governance rules: six pillars (Philosophy / Industry / Data-class / Roles / Controls / Review), data classifications, agentic-AI policies, and the compliance sections you attest against over time.
Upload your AI_Governance_Framework.md — not shipped with the platform. Uploads land in the central Fleet DB so every instance in the fleet pulls the same authoritative version automatically. Each upload auto-increments the version counter; older versions are preserved for audit.
Seed Framework parses the current document and populates the navigable section tree. Idempotent — existing attestations and change proposals stay attached; only missing rows get inserted.
